# Shuttr

Shuttr is a static, frontend-only photography learning demo. It lets learners adjust aperture, shutter speed, and ISO, then see simulated teaching effects immediately in the image viewer.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The app uses hash routing, so it is safe to deploy on static hosts such as Netlify, Vercel, Cloudflare Pages, or GitHub Pages.
